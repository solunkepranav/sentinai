# 📸 Best Results: Take Live Screenshots!

The live app is running. Screenshots are more impressive than generated images:

1. Start `run_sentinai.bat`
2. Open `http://localhost:8000`
3. Use **Win+Shift+S** (Windows Snip Tool)
4. Capture each page:
   - Dashboard (with charts loaded)
   - Data Ingestion (after "Load Demo Dataset")
   - AI Analysis (during animated pipeline)
   - SAR Report (while hovering a cyan phrase)
   - Audit Trail (after analysis)
   - Exported Report (click Export → screenshot new tab)
